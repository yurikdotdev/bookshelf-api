# Bookshelf API Documentation

install dependencies:

```sh
bun install
```

run:

```sh
bun dev
```

<http://localhost:3000>

## Endpoints

### API Specification

| Endpoint | Method | Parameters | Response |
| --- | --- | --- | --- |
| `/books` | GET | `page` | List of books |
| `/books/:id` | GET | `id` | A book |
